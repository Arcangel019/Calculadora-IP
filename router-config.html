Configuración de Router Cisco

Guía paso a paso para configurar tu router

Conectarse al router

Would you like to enter the initial configuration dialog? [yes/no]: no

Press RETURN to get started!

Router>

Ingresar al modo privilegiado

Router>enable

Router#configure terminal

Enter configuration commands, one per line. End with CNTL/Z.

Router(config)#

Cambiar nombre al router

Router(config)#hostname RedesI

RedesI(config)#

Configurar contraseñas

Contraseña para consola

RedesI(config)#line console 0

RedesI(config-line)#password 123456

RedesI(config-line)#login

RedesI(config-line)#exit

Contraseña modo privilegiado

RedesI(config)#enable secret 123456

Contraseña acceso remoto (VTY)

RedesI(config)#line vty 0 15

RedesI(config-line)#password 123456

RedesI(config-line)#login

RedesI(config-line)#exit

Configuraciones adicionales

Encriptar contraseñas

RedesI(config)#service password-encryption

Configurar banner

RedesI(config)#banner motd "ACCESO, SOLO A PERSONAL AUTORIZADO"

Configuración de interfaces

RedesI(config)#interface gigabitEthernet 0/0

RedesI(config-if)#ip address 192.168.1.1 255.255.255.0

RedesI(config-if)#no shutdown

// Realizar la misma operación por cada interface

Guardar configuración

RedesI#copy running-config startup-config
